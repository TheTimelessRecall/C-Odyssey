## Getting Started

I used a `Makefile` to keep things simple. No need to type `gcc` manually for every program.

1.  **Clone the Repo:**
    ```bash
    git clone https://github.com/TheTimelessRecall/C-Odyssey.git
    cd C-Odyssey
    ```

2.  **Compile the Code:**
    You can compile all programs at once or just a specific one.
    * **Compile all programs:**
        ```bash
        make all
        ```
    * **Compile a single program** (e.g., `bmi_cal.c`):
        ```bash
        make bin/bmi_cal
        ```
    All compiled programs will be placed in a `bin/` directory.

3.  **Run an Executable:**
    ```bash
    ./bin/bmi_cal
    ```

4.  **Clean Up (Optional):**
    To remove all compiled files, run:
    ```bash
    make clean
    ```
